## Pythonでの引き算

Pythonで引き算を行うときは、`-` 演算子を使います。基本形は `a - b` です。

標準入力から2つの整数を受け取って計算する例:

```python
a = int(input())
b = int(input())
print(a - b)
```

## Pythonでの足し算

Pythonで足し算を行うときは、`+` 演算子を使います。基本形は `a + b` です。

標準入力から2つの整数を受け取って計算する例:

```python
a = int(input())
b = int(input())
print(a + b)
```
