## 問題文

整数$A,B$が与えられます。
$A-B$の計算結果を出力してください。

## 入力データの特徴

- $0 \leq A,B \leq 10^9$
- 入力は全て整数である。

---

## 入力

入力は以下の形式で標準入力から与えられる。

```
$A$ $B$
```

## 出力

$A-B$を出力せよ。

---

## 入力例1

```
3 1
```

## 出力例1

```
2
```

---

## 入力例2

```
5 3
```

## 出力例2

```
2
```
