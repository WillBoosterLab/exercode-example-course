## 問題文

整数 $N$ が与えられます。
$N$ を 2 倍した値を出力してください。

## 入力データの特徴

- $0 \leq N \leq 10^9$
- 入力は整数である。

---

## 入力

入力は以下の形式で標準入力から与えられる。

```
$N$
```

## 出力

$N \times 2$ を出力せよ。

---

## 入力例1

```
4
```

## 出力例1

```
8
```

---

## 入力例2

```
15
```

## 出力例2

```
30
```
